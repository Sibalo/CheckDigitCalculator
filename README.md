# CheckDigitCalculator

# Serial Number Validator
The application accept an 8-digit serial number and calculate a check digit

## Features
- Accepts 8-digit serial number
- Calculates check digit
- Real-time validation

## Architecture
- WinForms UI
- Check digit logic in `CheckDigitService`
- TextBox input logic in `InputNavigationService`

## How to Run
1. Download the zip file
2. Extract the zip file
3. Open the solution(.sln)
4. Restore NuGet packages
5. Press Run
